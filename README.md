## Design Theme
Modern minimalistic developer portfolio showcasing my skills, projects, and contact information. The design uses a clean layout with a professional color scheme, responsive design, and simple typography.

---

## CSS Techniques Used

- CSS Custom Properties (Variables)  
- Flexbox (Navigation and Footer)  
- CSS Grid (About section and Projects section)  
- Media Queries (Mobile-first responsive design with 3 breakpoints)  
- Other: Smooth buttons, hover effects  

---

## Project Sections

1. **Home / Hero Section**  
   - Name, tagline, and call-to-action button linking to Projects.  

2. **About Section**  
   - Professional bio (150+ words)  
   - Profile image  
   - Skills list  

3. **Projects Section**  
   - 3 project cards with title, description, image, and links  

4. **Contact Section**  
   - Contact form with labels and validation attributes  

---

## Challenges & Solutions

- **Responsive Grid:** Initially the project cards stacked vertically on tablets; fixed by using `grid-template-columns: repeat(2, 1fr)` at 768px.  
- **Image sizing:** Profile image was too large; solved using fixed width and border-radius for a circular effect.  
- **Form accessibility:** Added `label` elements and `for` attributes to improve accessibility.
